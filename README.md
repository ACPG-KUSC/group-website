# Applied Cryptography and Privacy Group (ACPG)

This repository contains the source code for the official website of the **Applied Cryptography and Privacy Group (ACPG)** at Korea University, led by Prof. Seongkwang Kim.

## About the Lab
ACPG focuses on advancing the frontiers of cryptography, with main research areas including:
- **Symmetric-Key Cryptography**
- **Post-Quantum Cryptography**
- **Privacy-Preserving Protocols**

## Website Structure
- **Home**: Overview of the lab, research areas, news, and recruiting information.
- **Members**: Profiles of the principal investigator (Faculty) and graduate students.
- **Publications**: List of academic papers and preprints.
- **Contact**: Lab location, email, and Google map directions.
